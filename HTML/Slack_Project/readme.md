# Slack Landing Page Clone

A responsive Slack-inspired landing page built using pure HTML as part of my Sigma Web Development journey.

## Project Overview

This project recreates the structure and user experience of Slack's marketing website while focusing on semantic HTML, accessibility, SEO-friendly markup, and proper project organization.

## Features

* Semantic HTML structure
* Multi-page website architecture
* Responsive-ready layout structure
* Login page
* Signup page
* Navigation bar
* Hero section
* Features section
* Knowledge section
* Footer section
* Font Awesome icons
* Forms with validation

## Pages

### Home Page
